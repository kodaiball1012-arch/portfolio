Profile & Vision

テクノロジー（実装力）とビジネス（推進力）を高度に融合させ、次世代の広告プロダクトを牽引するProducer

エンジニアリングの現場で培った「複雑なデータからアルゴリズムを実装・最適化する技術力」と、営業の最前線で叩き上げた「データドリブンな目標達成力（Get Things Done）」を掛け合わせることで、AIやビッグデータを駆使した革新的な広告プロダクトを創出します。技術の限界と可能性をコードレベルで理解しているからこそ描ける、実現可能かつ圧倒的なスピード感を持った事業戦略を推進します。



Core Competencies

データドリブンな課題解決能力: 生体信号データから営業KPIまで、あらゆる事象を定量的に分析し、最短距離で最適解を導き出す仮説検証力。

泥臭い実行力（Get Things Done）: 論文調査、有識者へのヒアリング、日々のフィードバック収集など、現場の一次情報を即座に実装・行動へ反映させる高速なPDCAサイクル。

Tech × Businessの橋渡し: 暗号技術や機械学習の根幹となる数理・アルゴリズムの理解を、プライバシー保護やターゲティング精度向上といった直接的なビジネス価値へ翻訳する能力。

1. Machine Learning & Data Analysis (MATLAB)

生体信号（心電図）解析と機械学習を用いた分類アルゴリズムの実装

複雑な時系列データである心電図（ECG）に対し、Support Vector Machine（SVM）を用いた特徴量抽出と分類アルゴリズムをフルスクラッチで実装し、システムの有効性を検証しました。



課題とアプローチ

新規特徴量を導入した際、精度の頭打ちという壁に直面しました。この課題に対し、単なるAIツールの活用に留まらず、国内外の先行研究（論文）の徹底的な調査を実施。さらに、教授や先輩エンジニアとの泥臭い技術的ディスカッションを高速で回し、第三者の客観的なフィードバックを即座にコードへと反映させることで、アルゴリズムのボトルネックを突破しました。

実績・成果

研究室内の誰よりも早くプロジェクトを完遂し、システム制御情報学会（SCI）にて論文提出および優秀賞を受賞しました。

💡 Business Application (Producer視点)

「ノイズの多い複雑なデータから本質的な特徴量を見抜き、推論精度を極限まで高めるプロセス」は、広告プロダクトにおけるターゲティング精度の向上に直結します。膨大なユーザーの購買・行動履歴データから最適なセグメントを抽出し、広告費用対効果（ROAS）を最大化する配信アルゴリズムの設計・最適化において、このデータ解析の知見と実装経験を強力な武器として活用します。

2. Information Security & Cryptography (Python)

公開鍵暗号方式のプロトコル実装とセキュア設計

情報セキュリティの根幹を成す暗号技術に対し、理論の学習だけでなく、実際にPythonを用いてRSAやElGamalといった公開鍵暗号方式のプロトコルを自らコードに落とし込み、システムの暗号化・復号プロセスを実装しました。



💡 Business Application (Producer視点)

サードパーティCookieの廃止など、デジタル広告業界においてユーザーデータのプライバシー保護（データクリーンルーム等の活用）が急務となっています。データセキュリティの仕組みをブラックボックスにせず、コードレベルで理解していることは、コンプライアンスを遵守しつつもデータを最大限に活用できる、安全で信頼性の高い次世代広告プロダクトを企画・設計する上で、非常に強固な基盤となります。

3. Business Execution & Data-driven Sales

トップ3%の成果を叩き出した、データドリブンな営業インターン

買取専門店の営業インターンシップにおいて、100名中トップ3の成績を3ヶ月連続で達成しました。気合いや根性論ではなく、徹底した数値管理と最速の学習サイクルによって成果を創出しています。



データに基づく逆算思考

自身の「成約率」や「アポイント取得率」といったKPIをデータとして客観的にトラッキングし、月間目標の達成に必要なアクションを日々の行動レベル（KDI）まで精緻に逆算して実行しました。

高速なPDCAと現場主義（Get Things Done）

「成果を出している人間に学ぶのが最速」という原則のもと、店長に対し毎日のフィードバックを主体的に要求。自身の営業プロセスにおいて浮き彫りになった課題やボトルネックを翌日の営業活動に即座に反映させる、極めて泥臭くスピード感のあるPDCAサイクルを回し続けました。

💡 Business Application (Producer視点)

新規広告プロダクトの立ち上げや拡販において、目標達成への執念と、数値をベースにしたボトルネックの特定・改善能力は不可欠です。楽天の掲げる「スピード」と「実行力」を体現し、事業計画を確実に達成へと導く推進力として機能します。
